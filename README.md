## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Run API'S
Run json-server before use this app

```sh
$ npm run json:server
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```
